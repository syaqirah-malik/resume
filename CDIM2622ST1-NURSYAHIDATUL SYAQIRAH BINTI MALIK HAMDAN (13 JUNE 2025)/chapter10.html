Chapter 10: Formatting Text with Styles (CSS)

Choosing a Font Family

body { font-family: Geneva; } h1, h2 { font-family; "Gill Sans"; }

Explanation: all the h1 and h2 words will be displayed in Gill Sans, while the font-family will be displayed in Geneva fonts in the html file.

Specifyting Alternate Fonts

body { font-family: Geneva, Tahoma, sans-serif; } h1, h2 { font-family: "Gill Sans", "Gill Sans MT", Calibri, sans-serif; }

Creating Italics

body { font-family: Geneva, Tahoma, Verdana, sans-serif; } h1, h2 { font-family: "Gill Sans", "Gill Sans MT", Calibri, sans-serif; } p { font-style: italic; }

Explanation: all the words in the html will be displayed in italic.

Applying Bold Formatting

body { font-family: Geneva, Tahoma, Verdana, sans-serif; } h1, h2 { font-family: "Gill Sans", "Gill Sans MT", Calibri, sans-serif; font-weight: normal; } em, a:link, .intro .subhead { font-weight: bold; }

Explanation: the introduction at the very first/early page will be bold.
